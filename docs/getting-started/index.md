## What Maxx Adds

Maxx is an unofficial Ghostty fork that keeps the terminal core lightweight and
adds a visible control layer for coding-agent workflows: sidebar sessions, agent
status indicators, and optional tab-control skills.

For general terminal behavior and configuration inherited from Ghostty, use
[upstream Ghostty documentation](https://ghostty.org/docs). For Maxx-specific
agent workflows, use the docs in this repository.
